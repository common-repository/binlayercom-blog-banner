== Description ==

Adds an Adcode from Binlayer.com to your page as a widget. Publisher Account from <a href="https://binlayer.com/referral-215319.html" target="_blank">binlayer.com</a> is required.


== Installation ==

1. Upload 'binlayer' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the new Widget to your Page
